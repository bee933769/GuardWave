## About
GuardWave is a local-first security CLI for monitoring and logging suspicious events in real-time.
It supports PDF export, incident classification, and an intuitive dashboard.

## Installation
Run:
./install.sh

## Run the App
Run:
./bin/run.sh
# OR
python3 src/guardwave.py
